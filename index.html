Cool features:
  • The camera is smooth (try that by moving the fox)
  • It is possible to render textures and colors at the same time (and mixing them is possible -> take a look at the leaves)
  • It is possible to use any texture as long as it is a png file
  • The skybox is mapped to look "real"
  • The floor is composed of only one cube where the mipmap is repeated
  • Semi-transparent textures are rendered properly (because the transparent textures are sorted dynamically)
  • No z-fighting
  • Only the visible face is rendered for each cube (face culling)
  • The fragment lighting is working. It is possible to move the light, to change its color, and it gets updated according to the daytime (night/day)
If the console shwos the error "gl.INVALID_OPERATION in drawElements(gl.TRIANGLES, 36, gl.UNSIGNED_BYTE, 0)" then you need to turn on "hardware acceleration" in your browser.
